# 🏏 ipl-auction-simulator
A web-based IPL Auction Simulator built with HTML, CSS, Tailwind, and JavaScript where users can create teams, bid on players, and simulate a real-time cricket auction with purse tracking and player statistics.

The project simulates the auction environment with **live bidding, team purse tracking, player database, and auction controls**.

---

## 🚀 Features

- 🏏 Create up to **10 teams**
- 💰 Set **custom team purse values**
- ⚡ Animated **neon UI design**
- 📊 **Live purse tracker** for all teams
- 🎯 Player **bidding system with increments**
- - ✅ Mark players as **Sold / Unsold**
- ⏭ Skip players during auction
- 📋 **100+ international and domestic players**
- 🌍 Player filters by **country and role**
- 📱 Responsive design for different screen sizes

## 🎮 How It Works

1. Add teams with their **name and purse amount**.
2. Start the auction after adding at least **2 teams**.
3. A player will appear in the **Current Player section**.
4. Select a team and place bids using increment buttons.
5. Mark the player as:
   - ✅ Sold
   - ❌ Unsold
   - ⏭ Skip
6. The **team purse updates automatically** after purchase.
